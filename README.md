# Krovhal SWA (Sovereign Web Agent)

O **Krovhal** é um agente de inteligência digital focado em **Soberania e Privacidade**. 

Ao contrário de IAs convencionais, o Krovhal opera sob o princípio *Client-Side Total*, o que significa que o processamento e o armazenamento de dados ocorrem exclusivamente no dispositivo do usuário.

## Funcionalidades
- **Vault Local:** Armazenamento encriptado no navegador (LocalStorage).
- **Criptografia XOR:** Camada de segurança personalizada para proteção de dados sensíveis.
- **Varredura Tática:** Busca de informações em tempo real via Proxy Soberano.
- **Memória Seletiva:** O usuário decide o que merece ser assimilado pela base de conhecimento.
- **Extração/Injeção:** Portabilidade total dos dados via arquivos `.json`.

##  Tecnologias
- HTML5 / CSS3 (Interface de Terminal)
- JavaScript (Motor Lógico e Criptografia)
- AllOrigins API (Proxy de Varredura)
